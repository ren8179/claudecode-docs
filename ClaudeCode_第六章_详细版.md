### 三大核心功能回顾

#### 1. Skills（技能包）

**核心价值：** 预封装工作流，用完即走

**关键概念：**
- 渐进式加载（3个层级）
- 零代码创建
- 多技能联用

**使用场景：**
- 反复解释同一件事 → 创建Skill
- 需要特定知识材料 → 创建Skill
- 多流程协同任务 → 创建Skill

#### 2. MCP Servers

**核心价值：** 连接外部服务，避免手动复制粘贴

**常用MCP：**
- chrome-devtools-mcp：浏览器自动化
- github：GitHub API集成
- postgres：数据库操作
- context7：最新文档查询

**使用场景：**
- 需要访问外部服务 → 使用MCP
- 需要浏览器自动化 → Chrome DevTools MCP
- 需要GitHub集成 → GitHub MCP

#### 3. CLAUDE.md

**核心价值：** 项目记忆文件，AI进化机制

**关键特性：**
- 自动读取
- 团队共享
- AI进化

**使用场景：**
- 记录项目规范
- 记录技术栈
- 记录常用命令
- 记录架构决策

## 6.2 常见问题速解

### Q1：如何选择合适的模型？

| 模型 | 提供商 | 代码能力 | 价格 | 优势 | 适用场景 | 推荐度 |
|------|--------|---------|------|------|---------|--------|
| **GLM-4.7** | 智谱AI | ⭐⭐⭐⭐⭐ | 中等 | 中文理解强，有Coding套餐 | 中文项目为主 | ⭐⭐⭐⭐⭐ |
| **Kimi K2** | 月之暗面 | ⭐⭐⭐⭐⭐ | 较低 | 超长上下文128K | 大型项目重构 | ⭐⭐⭐⭐⭐ |
| **Qwen-Coder-Plus** | 阿里云 | ⭐⭐⭐⭐⭐ | 低 | 开源，性能优秀 | Python/JS项目 | ⭐⭐⭐⭐ |
| **DeepSeek-Coder** | 深度求索 | ⭐⭐⭐⭐ | 极低 | 价格优势 | 预算有限的场景 | ⭐⭐⭐ |

- 中文项目 → GLM-4.7

- 预算有限 → DeepSeek-Coder
- 开源需求 → Qwen-Coder-Plus

### Q2：上下文满了怎么办？

**上下文使用率警告：**

```
0-30%: ✅ 良好，继续使用
30-50%: ⚠️ 注意，考虑压缩
50-70%: ⚠️ 警告，建议压缩
70-90%: 🚨 危险，必须压缩
90-100%: 🔴 严重，立即处理
```

**处理方法：**

```bash
# 方法1：压缩对话（保留摘要）
/compact

# 方法2：清空上下文
/clear

# 方法3：复制-粘贴重置法
# 1. 复制重要内容
# 2. /compact 获取摘要
# 3. /clear 清空上下文
# 4. 粘贴关键信息回去

# 方法4：限定对话范围
# 一个对话只处理一个任务
```

### Q3：如何提高Claude Code的输出质量？

**10个最佳实践：**

1. **使用Plan模式** - 先规划后执行
2. **提供具体的描述** - 清晰和具体的要求
3. **创建CLAUDE.md** - 记录项目规范
4. **精确引用文件** - 使用@指定文件
5. **任务拆解** - 大任务拆成小任务
6. **添加约束条件** - 明确边界和要求
7. **提供参考示例** - 给出期望的格式
8. **增量式开发** - 完成一步再进行下一步
9. **及时反馈** - 告诉Claude哪里不对
10. **使用Skills和MCP** - 扩展Claude的能力

**示例对比：**

```bash
# ❌ 模糊描述
帮我做一个登录功能

# ✅ 清晰描述
请实现用户登录功能：
1. 创建登录页面（email + password）
2. 使用React + TypeScript
3. 使用Tailwind CSS美化
4. 验证邮箱格式
5. 密码最少8位
6. 登录成功后跳转到首页
7. 使用localStorage存储token
确保代码严格遵守CLAUDE.md中的规范
```

```
│     └── 记录解决过程                                     │
│         ↓                                               │
│  5. 预防再次发生                                        │
│     ├── 更新CLAUDE.md                                    │
│     ├── 分享解决方案                                     │

```

**常见错误及解决方案：**

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| `command not found` | npm全局路径未配置 | 重启终端或添加到PATH |
| `API key invalid` | API密钥错误 | 检查环境变量配置 |
| `Context full` | 上下文已满 | 使用/clear清空 |
| `Permission denied` | 权限不足 | 使用`--dangerously-skip-permissions` |
| `Module not found` | 依赖未安装 | 运行`npm install` |
